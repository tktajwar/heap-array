# Heap Tree Data Structure

A heap tree (implemented in dynamic array) written in C!

It can be switched between min heap and max heap by modifying the header file

For Max heap:

```h
#define HEAP_TYPE max
```

For Min heap:

```h
#define HEAP_TYPE min
```
